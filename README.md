### Cafe Sales Data Case Study
This project is an exploratory data analysis  case study on cafe sales data. The analysis focuses on profitability across different locations, payment methods, and menu items to identify key drivers of revenue. Using Python libraries like Pandas, NumPy, Matplotlib, and Seaborn, the study cleans and visualizes the dataset, uncovering insights such as which items contribute most to profits, how customer payment preferences affect margins, and which locations perform best. The findings highlight actionable opportunities for café management to optimize operations and maximize profitability.
